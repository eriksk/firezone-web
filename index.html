Introducting 'Firezone: Infinity' - A hardcore procedurally generated shoot 'em up for Windows, Linux, Mac and Xbox One

Features

At the core Firezone: Infinity is a vertical shoot 'em up for up to four players

  • Procedural Stages
  • Procedural Bullet Patterns
  • Daily Challenges
  • Multiple difficulty settings
  • 1-4 players locally
  • Global highscores

Platforms

Firezone: Infinity is coming out on several different platforms

  • Windows
  • Linux
  • Mac
  • Xbox One
